①課題名
Chatアプリー＞普段用

②課題内容（どんな作品か）
基本的なChatアプリ
チャット内容に対して質問をしてくれる

③アプリのデプロイURL
https://github.com/Shogoricci/chat-app.git

④アプリのログイン用IDまたはPassword（ある場合）
なし

⑤工夫した点・こだわった点
チャットが続いていくなかで画面が自動でスクロールして会話が続けられるように設定した。当初チャットアプリを作った時はこの点が機能していなかった
会話中に異なる系統の質問をした時に会話の内容をリセットできない問題があったので、会話をリセットしてトピックを切り替える設定にした

⑥難しかった点・次回トライしたいこと（又は機能）
⑤の会話中に画面の一部のみをスクロールして会話が続けられるようにするのに時間を要した。
